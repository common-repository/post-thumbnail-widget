== Description ==

Allow to publish **post thumbnails** on sidebar and on RSS.

**Other plugins by iworks**: [Tag list](http://wordpress.org/extend/plugins/tag-list/), [AutoReferrer](http://wordpress.org/extend/plugins/auto-referrer/), [flakuj](http://wordpress.org/extend/plugins/flakuj/).

== Installation ==

1. Upload `post-thumbnail-widget` to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Use widget or configure plugin.
